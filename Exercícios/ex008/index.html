Termos em itálitco

Para usar a forma semântica para escrever coisas em itálico , basta usar a tag EM ( valor semantico)

Marcador de texto

Você também pode marcar um texto usando a tag MARK

Texto deletado

Você pode usar o comando del para informar um texto excluido

Texto inserido

Você pode usar o comando ins para colocar textos inseridos (sublinhados) no seu texto , para dar uma maio ênfase

Texto sobrescrito

Use o comando sup para sobrecrever coisas , como é o caso de elevar um número a outro

EX : (sem sup ) = x2 + 3

EX : ( com sup ) = x 2 + 3

textos subescritos

Use o comando sub para escrever coisas abaixo de outras , como é o caso das formulas químicas

EX : ( sem o sub) H2O

EX : ( com o sub) H 2 O
